## Run Locally

Clone the project:

```bash
git clone https://github.com/willy-r/bookstore-api.git
```

Go to the project directory:

```bash
cd bookstore-api
```

Install dependencies:

```bash
npm install
```

Start the local server:

```bash
npm run dev
```
